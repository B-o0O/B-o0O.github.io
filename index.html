JavaScript 历史回顾

1989年，瑞士日内瓦的欧洲核子研究组织（CERN）的Tim Berners-Lee首次提出万维网的系统概念，1991年其又开发了HTML、URL、HTTP等现代Web技术基石。几年时间内，Web技术愈发火热，网页变得越来越大，当时用户普遍使用28.8kbit/s 的调制解调器上网，而网页全都是使用服务器解析渲染，即使只是做简单的表单验证，用户与服务器之间的往返通信时间也让人难以接受。1995年，Web领头公司Netscape(网景)要求工程师Brendan Eich开发一门客户端脚本语言来处理这种简单的数据验证，其仅用10多天就完成了第一款JavaScript的开发，取名LiveScript，后来公司为蹭Java热度，改名为JavaScript，所以JavaScript与Java的关系其实并不大。

JavaScript 是什么

JavaScript 实现主要由三部分组成ECMAScript、BOM、DOM。
ECMAScript是标准JavaScript，也是现在大家口中真正意义上的JavaScript，叫JavaScript只是历史习惯。之所以出现ECMAScript，是因为当年LiveScript发布后，微软公司紧随其后发布了类似的脚本语言JScript，LiveScript和JScript都是为两家公司各自浏览器Navigator(现在的Firefox前身)和IE（Edge前身）服务的，所以有不同的实现，这意味着有两个版本的JavaScript，随着业界的日益担忧，标准化JavaScipt提上日程。1997年，第 39 技术委员会花了几个月时间打造了 ECMA-262，才有了现在的标准JavaScript,ECMAScript。ECMAScript是作为一门标准语言来制定的，Web浏览器只是ECMAScript的一种宿主环境，其他宿主环境有node.js等，所以ECMAScript也可以用来写服务端程序。

BOM是浏览器对象模型（Broswer Object Model）,提供了操作浏览器的API，DOM是文档对象模型(Document Object Model)提供了操作DOM的API。
